Lyt-js

Simple JS library with commonly used functions witten in Native JavaScript

View project on GitHub

Available Functions :

lyt.find(selector, selectorType, parentReference, index)

Will find and return the html element from DOM

selector

Mandatory field. value should be paased as string and it can be name of Class/Id/TagName/Name or a css-query selector.

selectorType

Mandatory field. what kind of selector is passed. Possible values and return types. 'id': when element is to found by using id. will return the element with id passed in selector.

'class': when element is to found by using class. will return array of all the elements with class passed in selector.

'tag/tagname': when element is to found by using html-tag. will return array of all the elements with tag passed in selector.

'name':when element is to found by using name. will return element with name passed in selector.

emptystring('')/undefined:**pass '' or leave this field undefined when css-query is passed in selector. Will return array of all the elements with css-query passed in selector.

parentReference

Optional field. if not passed search will be made in entire document. else reference of and element can be passed within which search will be performed

index

Optional field. In case of class/tag/css-query, return type is an array. Pass an integer value to directly get the element instead of array.

example: 

`<div id="main-container">
    <span class="inner-content">1</span>
    <span class="inner-content">2</span>
    <span class="inner-content">3</span>
    <input type="text" name='username' />
</div>
<span class="inner-content">3</span>`

var ref = lyt.find('main-container','id'); // will return reference of "main-container"
var ref = lyt.find('inner-content','class'); // will return array of elements with class name "inner-content"
var ref = lyt.find('span','tag'); // will return array of all the spans
var ref = lyt.find('username','name'); // will return reference of input with name='username'

var parentRef = lyt.find('main-container','id'); var ref = lyt.find('inner-content','class', parentRef ); // will return array of elements with class name "inner-content" within the "main-container" i.e only 1, 2 and 3

var ref = lyt.find('inner-content','class', parentRef, 1 ); // will return reference of second element with class name "inner-content" within the "main-container" .
